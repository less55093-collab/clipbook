# Clipbook - 极简 Windows 剪贴板管理工具

![Python](https://img.shields.io/badge/Python-3.8%2B-blue))
![License](https://img.shields.io/badge/License-MIT-green)
![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey)

**Clipbook** 是一款专为 Windows 设计的现代化剪贴板管理工具。它摒弃了繁杂的功能，专注于轻量、美观与高效。不仅能无限记录你的复制历史，还支持对文本和图片进行即时编辑，是你提升生产力的最佳伴侣。

> 🎨 **设计理念**：在占用极少系统资源的前提下，提供最流畅的操作体验。

## ✨ 核心功能

*   **♾️ 无限历史记录**：不再担心丢失刚才复制的内容，Clipbook 会自动将历史记录持久化保存到本地数据库。
*   **📝 图文双修**：完美支持纯文本和图片（位图）的混合存储与预览。
*   **✏️ 内置编辑器**：
    *   **文本**：直接在历史记录中修改文字内容。
    *   **图片**：支持简单的图片预览与编辑功能。
*   **🚀 极速轻量**：
    *   优化的内存占用机制，后台静默运行不卡顿。
    *   智能压缩存储算法，占用极小的硬盘空间。
*   **🖥️ 开机自启**：配置简单，开机自动驻留系统托盘，随时待命。
*   **💎 极简美学**：精心设计的 UI 界面，小巧精致，完美融入 Windows 原生风格。
